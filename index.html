This is an experimental library that generates web forms for you.

  • Uses JSON Schema to build HTML5 form on the fly.
  • Provides an API for building custom form controls, e.g TreeView control
  • Optionally extends schema object in order to achieve better customisation.

Examples:

  1. Simple contact form (Bootstrap 4)
  2. Profile setting (Bootstrap 4)
    • Predefined model
    • Responsive tree view control
    • Combination of several sub-schemas
